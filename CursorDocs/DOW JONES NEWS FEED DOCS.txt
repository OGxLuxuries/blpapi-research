DOW JONES NEWS FEED DOCS
Introduction
Service Description
The Dow Jones News Feed Service delivers low-latency access to Dow Jones news content in machine-readable formats, including HTML, XML, and JSON, allowing you to access breaking news stories for use in your research and analytics applications.

Service Scope
API Endpoint
The Dow Jones News Feed Service is available via the Bloomberg API (BLPAPI).

The following API endpoint allows you to access Dow Jones News Feed data:

//blp/mktnews-dowjonesnews

Required EID
The Dow Jones News Feed Service is available over Bloomberg Real-Time Services at this EID:

81347

Ensure that your application and the users of the application are entitled to the service and all of the requested documents.

Available Data
The Dow Jones News Feed provides the following content:

Barron’s (weekly content)
Dow Jones News (Japanese)
Dow Jones News (English, German), WSJ Pro Verticals, Barron’s (real time), MarketWatch
Dow Jones News (Traditional Chinese)
Dow Jones News (Simplified Chinese)
Dow Jones Federal Filings Content
Dow Jones Press Releases Content
The New York Post
The Wall Street Journal (daily 2:00 am EDT run), containing all stories as they appear in the paper
Bulk Datasets
You can download end-of-day Dow Jones News Feed Bulk datasets at DATA <GO>. Before proceeding, you must have a valid Bloomberg Data License account with the appropriate permissions for downloading EDF data.

For more: Bulk Products > Dow Jones News

Subscribing to Data
 HINT

For detailed guidance on subscribing to real-time data using the Bloomberg API (BLPAPI), see Bloomberg API Guide > Subscriptions.

Subscription Strings
You can receive Dow Jones News Feed Service notifications in either XML or JSON format:

//blp/mktnews-dowjonesnews/eid/81347?format=xml
//blp/mktnews-dowjonesnews/eid/81347?format=json
Subscription Syntax
The subscription naming convention followed in the BLPAPI is as follows:

<service name>-<topic name>/<eid>?<options>

Convention
Description
<service name>
The name of the service to which you want to subscribe, e.g., //blp/mktnews
<topic name>
The name of the topic for which you want to receive notifications, e.g., -dowjonesnews (the Dow Jones News Feed)
<eid>
The entitlement identifier that your application has been assigned in the Entitlements Management and Reporting System (EMRS)
<options>
The format of the //blp/mktnews service data in the CONTENT element. The syntax is format=<format>, where the format is either xml or json.
